# 🏠 House Price Prediction App

This project predicts house prices using a Machine Learning model.

The application is built using Python and Streamlit and allows users to enter house details such as area, number of bedrooms and age of the house to estimate the price.

## 🚀 Features
- Predict house price based on house details
- Simple and interactive web interface
- Fast machine learning prediction

## 🛠 Technologies Used
- Python
- Streamlit
- Pandas
- NumPy
- Scikit-learn

## ▶️ Run the Project

Install dependencies:

pip install -r requirements.txt

Run the app:

streamlit run app.py
