# House Price Prediction App

This project predicts house prices using Machine Learning.
The application is built using Python and Streamlit and allows users to enter house details such as area, number of bedrooms and age of the house to estimate the price.

Technologies Used-
- Python
- Streamlit
- Pandas
- NumPy
- Scikit-learn

Project Features
- Simple and interactive UI
- Real time price prediction
- Machine learning model integration
